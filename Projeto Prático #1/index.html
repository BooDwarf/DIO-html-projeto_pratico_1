H1 ATÉ H6

(h1, h2, h3, h4, h5, h6)

(Voltar ao inicio)

As tags h1 até h6 (h1, h2, h3, h4, h5, h6) são tags utilzadas como headers para criar títulos, subtítulos, etc.


PARÁGRAFO (p)

(p)

(Voltar ao inicio)

Diferente dos h1 até h6, o p (paragráfo) serve para fazer parágrafos. Este texto é o mais comum que você utilizará, pois ele é o texto de tamanho "normal"


MARK

(mark)

(Voltar ao inicio)

O mark (mark) serve para destacar textos entre as tags com a cor amarela.


SMALL

(small)

(Voltar ao inicio)

O small (small) serve para deixar textos, entre as tags, pequenos.


ITALIC

(i)

(Voltar ao inicio)

O italic (i) serve para deixar o texto que está entre as tags, itálico.


UNARTICULATED

(u)

(Voltar ao inicio)

O unarticulated (ul) serve para sublinhar textos dentre as tags.


STRONG

(strong)

(Voltar ao inicio)

O strong (strong) serve para deixar textos entre as tags em negrito.


ORDERED-LIST

(ol)

(Voltar ao inicio)

A ordered-list (ol) serve para criar-se uma lista na qual a ordem é importante.


UNORDERED-LIST

(ul)

(Voltar ao inicio)

A unordered-list (ul) serve para criar-se uma lista na qual a ordem não é importante.


LIST

(list)

(Voltar ao inicio)

A list é para ser colocada dentro de uma de duas opções de tags, ol (ordered-list) ou ul (unordered-list) para listar os itens dentre a lista que é ol (ordered-list) ou ul (unordered-list).


ANCHOR

(a)

(Voltar ao inicio)

A anchor (a) serve para criar um atributo href, que pode ser utliizado para navegar dentro da página (como você pode fazer para ir a um item específico nesta lista, ou voltar para o topo desta lista). Ou até um hyperlink que te levará para outra página.


HORIZONTAL-RULE

(hr)

(Voltar ao inicio)

Serve para criar uma linha reta como visto abaixo.


SUBSCRIPT TEXT

(sub)

(Voltar ao inicio)

A subscript-text (sub) serve para fazer com que o texto entre as tags aparentam um carácter abaixo da linha "normal", como visto por exemplo na fórmula da água, H2O


SUPERSCRIPT TEXT

(sup)

(Voltar ao inicio)

Oposto do subscript-text (sub), o superscript-text (sup), serve para deixar o texto entre as tags aparentarem um carácter acima da linha "normal", como visto por exemplo, em números elevados, 24.


BLOCKQUOTE

(blockquote)

(Voltar ao inicio)

blockquote (blockquote)
Serve para destacar o texto que está sendo citado de outra fonte.


FONT

(font)

(Voltar ao inicio)

A font serve para mudar o tamanho, a cor, e fonte do texto entre as tags. É necessário atribuir as propriedades no início da tag.


DELETED

(del)

(Voltar ao inicio)

A deleted (del) serve para mostrar que o texto dentre as tags são informações "deletadas".


ABBREVIATION

(abbr)

(Voltar ao inicio)

A abbreviation (abbr) serve para mostrar o significado ou intenção por trás daquela palavra para dar contexto, por exemplo, caso a pessoa não entende a palavra.
